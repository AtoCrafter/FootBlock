# FootBlock

Minecraft MOD


## 追加するアイテム

### 足元サプライヤー

このアイテムを持っていると、プレイヤーの足元にブロックを生成し続けます。
スニーク中は生成を停止します。

右クリックで生成のオンオフを切り替えます。

#### レシピ

```
■■■
F☆F
■■■
```

- ☆: ネザースター
- F: 羽
